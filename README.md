## Paldz — Portfolio

Creative portfolio for **video editing**, **graphic design**, and **motion graphics**.

### Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

### Edit content

- **Profile + services + projects**: `src/content/portfolio.ts`
- **Pages**: `src/app/`

### Deploy

Recommended: Vercel. Set `NEXT_PUBLIC_SITE_URL` to your production URL for correct OG/metadata URLs.
